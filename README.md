# Geo-Intelligence News Analytics System

This project is a real-time news analytics platform developed using Python that collects global news data using NewsAPI and displays it on an interactive geographical map. The system performs sentiment analysis on news headlines and classifies them as Positive, Negative, or Neutral.

## Features
- Real-time news fetching using NewsAPI
- Interactive world map visualization using Folium
- Sentiment analysis using TextBlob
- Local timezone integration
- Bar chart and pie chart analytics
- Country and city-based news exploration

## Technologies Used
- Python
- Folium
- TextBlob
- Matplotlib
- Requests
- Pytz
- NewsAPI

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run:

python main.py
